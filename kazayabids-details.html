Project Description

• Online car marketplace designed for buying and selling vehicles.
• User-friendly interface for easy navigation and seamless browsing.
• Sellers can create detailed listings with information and images of their cars.
• Buyers can browse through a wide range of car listings using search filters.
• Auction feature allows buyers to participate and bid on desired vehicles.
• Communication tools enable buyers and sellers to negotiate prices and arrange transactions.
• Secure and reliable platform for safe buying and selling experiences.
• Streamlined process for listing, searching, and purchasing vehicles.
• Connects car enthusiasts and facilitates interactions within the automotive community.
• Provides a convenient and efficient solution for car buyers and sellers.

Outcomes
Kazaya Bids aims to revolutionize the car buying and selling experience by providing a user-friendly platform with comprehensive features and secure transactions. The integration of PHP and MySQL APIs ensures a reliable and efficient marketplace for users, facilitating successful transactions and empowering individuals in their car-related endeavors.

Tech Stack Used
Android App Development with Java
API Development with PHP
API Integration with Volley
Payment Gateway Integration with PayStack
