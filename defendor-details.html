Project Description

• Defindor is an Android project developed as a crypto wallet management app.
• The app allows users to manage multiple crypto wallets within a single platform.
• Users can associate an email address with each wallet to filter and organize incoming wallet-related emails.
• Defindor utilizes face detection technology as a secure login method instead of traditional email-password authentication.
• Upon login, Mirror Reality SDK for Android detects the user’s face and prompts them to input a one-digit password for added security.
• This unique combination of face detection and a one-digit password adsds an extra layer of protection to the user’s account.
The app simplifies the management of crypto wallets by centralizing them into one easy-to-use interface.
• Defindor aims to enhance security and convenience in managing and accessing crypto assets.
• By leveraging face detection and email filtering, the app offers a user-friendly and secure experience.
• Defindor provides peace of mind for users, ensuring that their crypto wallets are easily accessible yet protected from unauthorized access.

Outcomes
Defindor offers a convenient and secure solution for managing multiple cryptocurrency wallets within a single app. By incorporating face detection technology and a one-digit password, the app ensures the privacy and security of user accounts while providing a user-friendly interface for efficient wallet management.

Tech Stack Used
Android App Development with Kotlin
Android Mirror Reality face recognition
API Integration with Retrofit
MVVM
Payment Gateway using Google Pay
